# Well-Orderings of the Natural Numbers

[Website](https://hexanna1.github.io/well-orderings/)

A collection of [explicit well-orderings](https://hexanna1.github.io/well-orderings/well_orderings.html) of the natural numbers,
from Cantor normal form to extended Buchholz notation. Each encoding gives a bijection from natural
numbers to ordinal notations, additive with respect to Hessenberg natural sum.

Separately, the [ordinal notation explorer](https://hexanna1.github.io/well-orderings/explorer.html) lets you navigate the notation
systems via fundamental sequences.

## Encodings

| Encoding | Notation | Limit |
| --- | --- | --- |
| [Cantor](https://hexanna1.github.io/well-orderings/well_orderings.html#cantor) | Cantor | $\varepsilon_0$ |
| [Binary](https://hexanna1.github.io/well-orderings/well_orderings.html#binary) | Binary Veblen | $\Gamma_0$ |
| [Finitary](https://hexanna1.github.io/well-orderings/well_orderings.html#finitary) | Finitary Veblen | small Veblen ordinal |
| [Transfinitary](https://hexanna1.github.io/well-orderings/well_orderings.html#transfinitary) | Transfinitary Veblen | large Veblen ordinal |
| [Buchholz₁](https://hexanna1.github.io/well-orderings/well_orderings.html#buchholz_1) | Buchholz (one uncountable) | Bachmann-Howard ordinal |
| [Buchholz](https://hexanna1.github.io/well-orderings/well_orderings.html#buchholz) | Buchholz ($\omega$-many uncountables) | Buchholz ordinal |
| [EBOCF](https://hexanna1.github.io/well-orderings/well_orderings.html#ebocf) | Extended Buchholz | $\psi_0(\Lambda)$ |
