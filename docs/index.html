A collection of explicit well-orderings of the natural numbers, from Cantor normal form to extended Buchholz notation. Each encoding gives a bijection from natural numbers to ordinal notations, additive with respect to Hessenberg natural sum.

Separately, the ordinal notation explorer lets you navigate the notation systems via fundamental sequences.

Cantor: Read prime factorization hereditarily as Cantor normal form. A factor \(p_m^k\) contributes \(\omega^\alpha\cdot k\), where \(\alpha\) is obtained by decoding \(m\) recursively. Collect the resulting finite sum in Cantor normal form.

Binary: Read prime factorization hereditarily as Veblen normal form. A prime-index node uses the odd/even prime-color split to decode a raw argument \(\gamma\) and Veblen index \(\beta\), giving \(\varphi_\beta(\gamma)\). Use the canonical finite-tail repair for fixed-point arguments excluded by Veblen normal form.

Finitary: Starting from Binary, replace its odd/even prime-color split by countably many prime colors \(p_{2^j(2i-1)}\), ordered by finite Veblen argument slots from the active/rightmost slot outward. Read the resulting finite argument strings in small-Veblen normal form, with the same active finite-tail repair.

Transfinitary: Starting from Finitary, replace finite argument colors by finite matrix-column atoms. Each atom uses the Binary odd/even prime-color split to decode an ordinal position \(\iota\) and a principal-term index for the value above \(\iota\). Group atoms with the same position into the natural code of that value, then read the finite-support matrix in large-Veblen normal form, with the same active-column finite-tail repair.

Buchholz₁: Decode a raw hereditary \(\psi\)-forest with two levels, \(\psi_0\) and \(\psi_1\), for the one-uncountable Buchholz fragment. For each level \(\nu\), scan the raw forest sequence and retain exactly the terms admissible as \(\psi_\nu\)-arguments. Decode the final natural number by finite decreasing sums of retained \(\psi_0\)-terms.

Buchholz: Starting from Buchholz₁, replace the two raw \(\psi\)-levels by all levels \(\psi_\nu\) with \(\nu<\omega\), using prime colors \(p_{2^\nu(2m-1)}\), where \(m\) gives the hereditary argument. For each finite level \(\nu\), scan the raw forest sequence and retain exactly the terms admissible as \(\psi_\nu\)-arguments. Decode the final natural number by finite decreasing sums of retained \(\psi_0\)-terms.

EBOCF: Starting from Buchholz, let the \(\psi\)-levels themselves be ordinal terms. Decode a raw hereditary forest by splitting each prime index into raw level and argument codes, giving raw terms \(\psi_\nu(\alpha)\). For each legal level \(\nu\), scan the raw forest sequence and retain exactly the admissible arguments at that level. Decode the final natural number by finite decreasing sums of retained \(\psi_0\)-terms.
